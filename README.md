
# React Blog Application

Simple blog implementation using TailwindCSS, React, ReactQuery and Vite.


## Running local

Clone project

```bash
  git clone git@github.com:Bilusca/react-blog.git
```

Install dependecies

```bash
  yarn
```

Running app

```bash
  yarn dev
```

## Stack

**Front-end:** React, Vite, tailwindCss, React Query.
